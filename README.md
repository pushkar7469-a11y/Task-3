# Task-3
📚 Student Record Management System in C
📌 Project Overview

The Student Record Management System is a menu-driven application developed in C programming to manage student information efficiently. It demonstrates the use of structures, functions, arrays, and file handling (if implemented) to perform basic CRUD (Create, Read, Update, Delete) operations on student records.

This project is ideal for beginners who want to understand how real-world applications can be built using core C programming concepts.

🚀 Features
➕ Add a new student record
📋 Display all student records
🔍 Search for a student by Roll Number
✏️ Update existing student information
🗑️ Delete a student record
📌 Menu-driven interface for easy navigation
🧩 Modular programming using user-defined functions
🛠 Technologies Used
Programming Language: C
Compiler: Dev-C++ / GCC
Concepts: Structures, Functions, Arrays, Strings, Conditional Statements, Loops
📖 Program Explanation

The program uses a structure (struct Student) to store student information.
